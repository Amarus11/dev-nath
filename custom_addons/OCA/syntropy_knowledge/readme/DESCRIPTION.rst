Syntropy Knowledge Base
========================

Módulo de gestión del conocimiento organizacional para Odoo 18.

Características principales:

* Artículos con editor HTML enriquecido y colaborativo
* Categorías jerárquicas con iconos visuales
* Etiquetas para clasificación transversal
* Sistema de favoritos personales
* Flujo de publicación: Borrador → Revisión → Publicado
* Versionado automático al publicar con opción de restaurar
* Control de acceso por roles (Lector / Editor / Administrador)
* Vistas Kanban, Lista y Formulario
* Integración con chatter (mail.thread) para colaboración
* Soporte multi-compañía
* Contador de vistas por artículo
* Búsqueda avanzada por contenido, categoría, etiqueta y autor
